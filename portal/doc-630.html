4. 엔드투엔드 데이터 파이프라인

[유형별 입력 어댑터]  원고=(직접) / 영상=STT+OCR+영상분석 / 상품=OCR   ← Post-MVP 확장
        ▼ (텍스트/태그로 정규화)
규정·판례 원천 → 지식화(온톨로지·벡터 인덱싱) → 검색(규정매핑·판례 RAG)
        → LLM 판정 엔진(추론·근거 생성) → 판정초안 → 사람 확정
        → 심의이력 DB 적재 → 정확도·판정기준 보강 루프(→ 온톨로지·인덱스 환류)

MVP는 어댑터 없이 원고 텍스트가 바로 지식화·검색 단계로 진입.

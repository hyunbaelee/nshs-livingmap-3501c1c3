1. 개요

방송 전 심의. 심의 대상을 접수·분류하고, 규정 적합성을 AI가 선분석·판정초안, 사람이 최종 확정해 제작·편성에 통보. MVP는 원고·기획(텍스트) 심의에 집중.

2. 전체 흐름 (Sub 3묶음 + 유형 축)

[5.1.1 접수·완결성·유형분류]
      │  (원고 / 영상 / 상품표시 라우팅)
      ▼
[5.1.2 규정 적합성 분석·판정]  ← 유형별 입력 어댑터(NLP/STT·OCR/OCR) + 공통 판정엔진
      │  선분석 → 신뢰도 라우팅 → 자동초안 / 근거보강 / 에스컬레이션
      ▼
[5.1.3 검토·확정·통보]  ← 사람 최종 확정 → 통보 → 이력·지식 적재
      │
      └──▶ 5.4 판정기준·지식 (보강 루프)

5. 공통 원칙

  • AI 보조 + 사람 확정 — 고위험·신유형 자동확정 금지, 최종 책임=사람.
  • 모든 판정·근거·신뢰도·모델버전 감사 로그.
  • 근거(조항·판례) 미첨부 초안 무효.

6. 연계

  • 5.4 판정기준·지식(온톨로지·RAG) → 5.1.2 AI 판정 근거 공급 + 5.1.3 이력 → 5.4 보강.
  • 5.2 사후심의 / 5.3 대외대응 = MVP Out-of-scope (PRD 60).
  • PRD: 60 심의 AI PRD ⧉ U-1외부~U-5외부 ↔ 5.1.2 액티비티.
